# Winery Dates

A directory web app for finding great wineries for date night, with filters for vibe and drive time plus a planning area for saved picks.

## Local development

```bash
nvm use
npm ci
npm run dev
npm run typecheck
npm run lint
npm run test
npm run build
```

## MVP status

- Directory page with search, vibe filter, and max-distance slider
- Starter winery dataset in `src/data/wineries.ts`
- Planner section for saved wineries and preferences
